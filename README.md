# hostlos.sh

## how to use
bash hostloc.sh username password times

### ex
bash hostloc.sh fdasv fdsdfsa 20

### cron
bash /opt/hostloc/hostloc.sh fdasv fdsdfsa 20 >/dev/null 2>&1 &


# tp
